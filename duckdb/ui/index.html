Available Endpoints

  • Health Check: GET /health
  • Service Info: GET /info
  • Statistics: GET /stats
  • List Documents: GET /ocr/documents
  • Execute Query: POST /query
  • Search Text: POST /search/text

Quick Example

Execute a SQL query:

POST /query
Content-Type: application/json

{
  "query": "SELECT filename, COUNT(*) as pages FROM ocr_pages GROUP BY filename",
  "limit": 100
}

Installing DuckDB-Wasm UI

To enable the interactive DuckDB UI:

  1. Download the latest DuckDB-Wasm bundle from github.com/duckdb/duckdb-wasm
  2. Extract the files to the ui/ directory
  3. Restart the service
